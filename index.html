ⓘ How to read this

The windows. “This week” and “Last 5 hours” are rolling usage windows that count everything you send to Claude — across every chat, task and tool, not just one conversation. Each one refills at the reset time shown.

How accurate. Token counts are exact — read straight from Claude Code's own logs. Percentages are approximate (shown with “≈”): Anthropic doesn't publish the exact limits, so we take the official percentage Claude reports and sharpen it with your exact token counts.

Per-model %. Each model shows how much of the limit that actually constrains it you've used — its own weekly cap if it has one, otherwise its use of the shared window quota. Not a share of tokens.

This machine vs your account. The ≈percentages are account-wide — they're the official figures, so they include every device, the web app, and mobile. The token counts cover only this machine (what its local logs contain). If you also use Claude or Codex elsewhere, the percentage will read higher than these token counts alone would suggest — that's expected, not a bug.

Codex. If OpenAI's Codex CLI is installed, its usage appears the same way — exact tokens from ~/.codex/sessions, official percentages from your own ChatGPT login.

Private. Everything is computed locally from ~/.claude and ~/.codex; nothing leaves your machine.
